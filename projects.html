Music Player

A music player made using HTML, CSS, JavaScript, and the YouTube player API. This web application features three Hindi songs, buttons to pause/play songs, move to next song, and also move to the previous song. It also features a progress bar at the bottom so that users can track the progress of the current song.

Shopping Center Website

This web application is a full stack website built using React/ Vite, JavaScript, and CSS for the frontend. For the backend, Flask, SQLAlchemy, and Python were used. The web application features a home page that displays trending, classical, pop, and music sections. In each of these sections, corresponding art pieces are displayed. Visitors to the website can click any one of the shown artworks to learn more about them and to also buy at varying quantities. Once they click the buy button, they will be led to the checkout page, where they can checkout artworks. The users can also create an account in the website so as to post artworks themselves to sell. If you are highly curious, please follow the instructions after clicking the image to learn how to view the website personally.
